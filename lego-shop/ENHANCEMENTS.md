## How to Use

### Use Enhanced Version
```html
<!-- In index.html -->
<link rel="stylesheet" href="styles-enhanced.css" />
<script src="script-enhanced.js" defer></script>
```

### Switch Back to Original
```html
<!-- In index.html -->
<link rel="stylesheet" href="styles.css" />
<script src="script.js" defer></script>
```
